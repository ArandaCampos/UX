Disponibilizar técnicas e animações de componentes web

### Como usar

Basta copiar os conteudos entre `<body> </body>` do `index.html`, e os arquivos `style.css` e `animation.js`, caso exista, no seu projeto.

### Tecnologias

- CSS3
- JS
